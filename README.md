# Packet Tracer Labs — CCNA 1 Introduction to Networks

Simulace síťových topologií vytvořené v Cisco Packet Tracer. Všechny tyhle simulace jsem stavěl a konfiguroval během školního roku ve Štětí.

---

## Soubory

| Soubor | Topologie | Popis |
| :--- | :--- | :--- |
| **franc-routing-basic.pkt** | 1 router, 3 switche, 8 PC | Základní konfigurace — hesla, Telnet, default-gateway, VLAN1 |
| **franc-routing-basic-v2.pkt** | 1 router, 4 switche, 12 PC | Rozšířená stromová topologie |
| **franc-routing-rip.pkt** | více routerů | Dynamické směrování pomocí protokolu RIP |
| **franc-routing-static.pkt** | 2 routery | Statické směrování |
| **franc-routing-static-v2.pkt** | 4 routery, 5 switchů, 10 PC | Statické směrování — komplexní mesh topologie, IPv4 + IPv6 |
| **franc-routing-static-default-gateway.pkt** | — | Konfigurace default gateway na přepínačích |
| **franc-routing-ipv6.pkt** | — | Topologie s IPv6 adresováním |

---

## Použité technologie

*   Cisco Packet Tracer
*   Cisco IOS — ISR1331, 2960-24TT
*   Statické směrování (ip route)
*   Dynamické směrování — RIP
*   Konfigurace switchů — VLAN, Telnet, hesla, default-gateway
*   Subnetting IPv4
*   IPv6
